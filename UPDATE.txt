Manual to move from version v2.0.0 to v3.0.0

1. Run "npm update".
2. Update the version.json file.
3. Update file "README.md".
4. New file "main.js".
5. New file "server.js".
6. New folder "sys-express".
7. File: package.json

	* The following "script" items are added or modified.
		"start": "node main.js",
		"grunt": "grunt",
		"server": "node server.js",
	* New packages: 
		- npm i express --save-dev

Ok, updated project.
